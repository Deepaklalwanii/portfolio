Guessing Game

I created an interactive Number Guessing Game where players have 5 attempts to guess a randomly generated number. For each correct guess, the player earns 1 point, while the computer earns a point for each incorrect guess. The current score is displayed on the scoreboard in real time, allowing players to track their progress. The game provides feedback after each guess, indicating whether the guess is too high, too low, or correct. Built using HTML, CSS, and JavaScript, this project showcases my ability to develop engaging, interactive web applications with dynamic functionality and real-time updates.
